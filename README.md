# Mieasystore
这是一个仿小米商城的小项目
1. 工程化项目
2. swiper轮播的使用
3. Ajax数据请求渲染DOM
4. 放大镜
5. QQ在线客服

## 主要功能：

1. 登录注册
2. 轮播图及其一二级导航
3. 商品页面数据渲染
4. 商品详情页面
